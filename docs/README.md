# WebShell Blog - 功能说明文档

## 1. 项目概述

WebShell Blog 是一个创新的终端风格博客系统，将传统 Linux 终端的简洁与现代博客的丰富内容完美结合。用户可以通过命令行交互方式浏览、阅读文章，享受独特的"黑客风"体验。

### 1.1 技术栈

- **前端**: Xterm.js + Vanilla JavaScript
- **UI 主题**: Snazzy Theme (Dracula 风格变体)
- **后端**: Go + WebSocket (可选，本版本为纯前端)

### 1.2 特性

- 真实的终端体验（命令输入、Tab 补全、命令历史）
- Linux 标准命令兼容
- TUI 风格双面板布局
- 终端内嵌图片显示
- 响应式设计

## 3. 命令详解

### 3.1 文件操作命令

#### ls - 列出目录内容

```bash
ls              # 简略列表
ls -l           # 长格式列表（显示权限、大小、日期）
ll              # 等同于 ls -la
ls -la          # 显示所有文件（包括隐藏文件）
```

**示例输出：**
```
$ ls
articles/  posts/  readme.md

$ ll
total 3
drwxr-xr-x   2 user  staff   128 Jan 15 10:00 articles/
drwxr-xr-x   2 user  staff   128 Jan 15 10:00 posts/
-rw-r--r--   1 user  staff   512 Jan 15 10:00 readme.md
```

#### cd - 切换目录

```bash
cd /home            # 切换到主目录
cd articles          # 进入 articles 分类
cd 前端技术          # 进入中文目录名
cd ..               # 返回上级目录
cd ../posts         # 返回上级并进入 posts
cd ../../           # 返回两级
cd ~                # 返回主目录
```

#### cat - 显示文件内容

```bash
cat readme.md        # 显示 README
cat 1.md             # 在终端显示文章内容
cat articles/         # 显示子目录列表
```

#### pwd - 显示当前路径

```bash
$ pwd
/home/articles/frontend
```

### 3.2 文章阅读命令

#### read - 阅读文章（预览模式）

```bash
read 1               # 在预览面板打开 ID 为 1 的文章
read 2               # 打开 ID 为 2 的文章
```

| 命令 | 说明 | 示例 |
|------|------|------|

## 4. Tab 自动补全

### 4.1 功能说明

| 场景 | 操作 | 效果 |
|------|------|------|
| 空输入 | 按 Tab | 显示当前目录内容 |
| 命令输入 | `ls<Tab>` | 自动补全 `ls -l ` |
| 目录补全 | `cd a<Tab>` | 补全为 `cd articles` |
| 中文目录 | `cd 前<Tab>` | 显示匹配的中文目录 |
| 相对路径 | `cd ..<Tab>` | 显示上级目录可选内容 |
| 文章 ID | `read <Tab>` | 显示可选文章 ID |

### 4.2 补全规则

- 命令名自动补全（输入 `h<Tab>` → `help `）
- 目录名智能补全（支持中文）
- 相对路径补全（`../`, `../../`）
- 参数选项补全（`-l`, `-la`）

## 6. 界面交互

### 6.1 终端与浏览器联动

- **左侧终端 → 右侧浏览器**：执行 `cd` 命令后，右侧自动切换到对应目录
- **右侧浏览器 → 左侧终端**：点击目录后，左侧终端自动执行对应命令

### 6.2 文章预览

1. 在终端执行 `read <id>` 或点击文章
2. 预览面板从右侧展开
3. 显示文章标题、元信息、内容和图片
4. 点击 `[x] close` 关闭预览
