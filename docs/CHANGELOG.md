## v1.0.0 (2024-03-22)

### 初始版本发布

#### 新功能

##### 1. 核心终端功能
- ✅ Xterm.js 终端模拟器集成
- ✅ 命令输入与历史记录
- ✅ 命令历史浏览（↑↓ 箭头键）
- ✅ Tab 键自动补全

##### 2. Linux 命令兼容
- ✅ `ls` - 目录列表（支持 `-l`, `-la` 选项）
- ✅ `ll` - 详细列表（等同于 `ls -la`）
- ✅ `cd` - 目录切换（支持相对路径 `../`, `../../`）
- ✅ `pwd` - 显示当前路径
- ✅ `cat` - 文件内容查看
- ✅ `clear` / `cls` - 清屏
- ✅ `echo` - 文本输出（支持 `-e` 转义）
- ✅ `whoami` - 当前用户
- ✅ `date` - 日期时间
- ✅ `uname` - 系统信息
- ✅ `history` - 命令历史
- ✅ `help` / `man` - 帮助文档

##### 3. 博客功能
- ✅ 文章分类目录结构
- ✅ `read <id>` - 文章预览
- ✅ 文章列表展示
- ✅ 文章内容渲染（支持图片）

##### 4. 双面板布局
- ✅ 左侧终端面板
- ✅ 右侧文件浏览器
- ✅ 左右联动（终端命令 ↔ 浏览器同步）

##### 5. 中文支持
- ✅ 中文目录名支持
- ✅ 中文 Tab 补全
- ✅ 退格键正确删除中文

##### 6. UI/UX
- ✅ Snazzy 主题配色
- ✅ TUI 风格界面
- ✅ 状态栏显示
- ✅ 面包屑导航
- ✅ About 页面

#### 技术实现

| 模块 | 技术方案 |
|------|----------|
| 终端 | Xterm.js 5.3.0 |
| 布局 | CSS Flexbox + Grid |
| 主题 | Snazzy (Dracula 变体) |
| 架构 | 纯前端 SPA |

## 贡献指南

### 开发环境

```bash
# 前端开发
cd web
# 直接打开 index.html 或使用静态服务器

# 后端开发 (Go)
cd server
go run cmd/main.go
```

### 代码规范

- JavaScript: ES6+ 标准
- CSS: BEM 命名规范
- 颜色: 使用 CSS 变量
